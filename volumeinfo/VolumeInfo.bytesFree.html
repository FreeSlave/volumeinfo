    Function VolumeInfo.bytesFree

    Free space in a volume

    long bytesFree() nothrow @property @trusted;

    Note

    This is size of free space in a volume, but actual free space available for the current user may be smaller.

    Returns

    number of free bytes in a volume or -1 if could not determine the number.
